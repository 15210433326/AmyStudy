#### VUE
  1. 安装 vue

    * 安装镜像  全局镜像在c 盘安装
      npm install -g cnpm --registry =https://registry.npm.taobao.org
      win10,win10 下的命令(提高权限)
      sudo npm install -g cnpm --registry =https://registry.npm.taobao.org

    * npm install -g cnpm 
    * 安装vue 运行环境
      cnpm install vue

    * 安装vue 脚手架
      cnpm install vue-cli

  2. vue js文件
    https://unpkg.com/vue/dist/vue.js
