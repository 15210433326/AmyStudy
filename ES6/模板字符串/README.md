#### 模板字符串
1. ES6模板字符串
  * 使用反引号 (数字1前面的按键)
  * 可以当做普通 字符串使用
  * 字符串如果嵌入变量，将变量名写在${中}
  ${}中
    ```javascript
      let name = 'tom'
      let str =  `hello ${name}`;
    ```
  * 可以定义多行文本
  ```javascript
    let str =` this i
    Tom`;

    console.log(str);

  ```
